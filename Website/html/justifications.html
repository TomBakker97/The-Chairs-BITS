Risks

Risk 1: VR function is not stable
Description: The VR function are very lag, and the VR function cannot be used on some older version of android.
Mitigation: We disabled the VR function if the user using version older than android 10, and this will give us more resource to develop the VR model and provide the best performance.

Risk 2: Error of the voice assistant
Description: The voice assistant function cannot provide the best solution to the user’s problem.
Mitigation: We input more data into the AI, so the AI can learn from these data and provide better solution to the user.

Risk 3: Compatibility between IOS and Android.
Description: The IOS system are not compatible with the Android system.
Mitigation: We decided to make an android app to reduce the cost and time, and this will give us more resource to develop the VR model and provide the best performance.
