Hows of the project

Resources and Tools

HTML5:
HTML5 is a language used for providing a structure and presenting contents on the web. We're using HTML5 because it's one of the most common languages for web development and we all have some experience using that language, the cost is also free. There isn't an alternative to HTML as almost programmer use HTML for web development

CSS:
CSS (Cascading Style Sheets) is a style sheet language that uses for styling the presentation of a document, image and video on the website, cost is free.

mySQL:
MySQL is a database management system. We are going to use mySQL for storing and managing data

Group Contract

Expectations around meetings

All members are expected to attend all meetings and need to notify other members when they cannot, exceptions apply such as family emergencies etc.

Assignment of work

All group members are expected to assign themselves a reasonable workload via trello so that no members are left doing more or less than others.

Process of unmet expectations

Members that are found to be not doing what's expected will be talked to, to try and help them understand what is neccessary, if no improvments are made then a poor sparkplus review should be expected.

Method for peer feedback

Messages and team meetings will be used for feedback

Frequency of responses

A message is expected to be replied to within 24 hours of its sending.

Tools used in the context of the weekly workflow

The team will be using Microsoft teams as our primary form of communication in which we have attached a trello extension to help us keep track of who's doing what on a week to week basis.
