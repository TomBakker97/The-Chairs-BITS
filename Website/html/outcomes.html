Project Motivation

What motivated our group to choose such a project was because of how sports such as soccer were cancelled due to the circumstances which have affected the world. Due to this reason we had come up with the idea to create and design an application where people would be able to see the statistics of soccer teams and players using Virtual Reality (VR). We believe that since people were/are unable to expereience these matches, by using VR it would provide a similar and intriguing experience for users as it would make it seem as if they were looking at a player in person.

How does it relate to your individual and collective interests

This project relates to our individual and collective interests as three of the members: John, Matthew and Nathan all have an interest in sports, some being soccer. This would allow for a wider range of ideas for the app as these members would have an understanding of what the targeted users would want to see. Hon's interest in cybersecurity and Thomas's interest in programming would both relate to and benefit the coding behind the app as the code itself is very essential for the app to work. As most of the group members have a similar level of programming to each other, by working on this project, it is expected that by the end of it, their confidence in their skills for IT would be strengthened as we are regularly working on software such as GitHub.

Features

Minimal features

  • The application produced will allow for users to use a navigation bar at the bottom of the screen to move to different screens and tabs of the application. This will be validated when the app is developed, and the navigation bar is placed and functioning taking the user to different places on the screen.
  • We will produce an application that allows the user to access a burger menu for the user to navigate and access different options of the application. This will be validated when the application has a burger menu displayed within the app containing different navigation options, settings and profile for user to access.
  • The application that we produce will contain voice assistance. The voice assistance will allow users to talk into the phone and have the players, teams, leagues etc. that they say to be recognised by the app and automatically search up what they had been looking for. This will be validated when the app has a button next to the search with a microphone icon to indicate that the app can recognise and assist with voice.
  • We will produce an application that allows for users to create accounts within the app to have custom settings and preferences. The profile will allow user to create a custom profile that allows them to have personal preferences on which clubs they would like to follow or favourite. This will be validated when the profile of the user I s created at the opening of the app and the user has their favourite clubs selected with information about those clubs given priority.

Extended features

  • We will produce an app that can compare different clubs and players against each other. The app will have a tab for compare where players and teams can be selected to compare. This will be validated with two players having their stats displayed side by side next to each other.
  • The application produced will be utilize VR to display player stats. The app will offer users to see player stats through VR interacting with player or team in a VR scenario. This will be validated with the option of VR given to users when looking across stats and displaying the “world”.
  • We will produce an application that allows for favourite clubs and players to be selected and prioritized. The app will have a favourites tab where it displays selected teams and players first on the dashboard of the application. This will be validated by having those teams and players selected as favourites and displayed in their specific tab and dashboard.
  • The application produced will contain iconic players of the past. The app will display statistics of iconic players of the football to be looked at and compared to other players. This will be validated when the player search has the option for icon players and contains those said players.
